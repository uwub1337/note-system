# 筆記管理系統本地端成品

這是 JavaScript 期末專題的本地端展示版。直接開啟 `index.html` 就可以操作，不需要登入、安裝後端、資料庫或套件。

## 已完成功能

- 筆記列表：顯示所有筆記，置頂筆記會排在前面
- 新增筆記：標題必填，分類預設為「未分類」
- 編輯筆記：選取列表中的筆記後可修改內容
- 刪除筆記：刪除後列表會立即刷新
- 搜尋筆記：支援搜尋標題與內容
- 分類篩選：依單一分類篩選筆記
- 收藏與置頂：可用於展示加分功能
- 本地端保存：資料會儲存在瀏覽器的 `localStorage`
- 展示資料：可一鍵重置示範筆記
- 主題顏色：可在暖木色、森林綠、海洋藍、玫瑰粉、石墨灰之間切換並保留偏好
- 匯出筆記：可將目前篩選結果匯出成文字檔

## 四人分工對應

- A 前端整合：`index.html` 與 `styles.css`，負責登入頁、列表頁、編輯頁與整體視覺
- B 筆記功能：`app.js` 裡的新增、編輯、刪除與表單驗證
- C 搜尋分類：`app.js` 裡的搜尋、分類、收藏、置頂與篩選
- D 資料測試：`app.js` 裡的 `localStorage`、展示資料、重置資料與整合測試

## 展示流程

1. 開啟 `index.html`
2. 新增一筆筆記
3. 編輯剛剛新增的筆記
4. 使用搜尋欄找筆記
5. 使用分類篩選
6. 勾選收藏或置頂
7. 刪除一筆筆記
8. 重整頁面，確認資料仍保留
9. 切換主題顏色，或匯出目前筆記

## 必測情境

- 空標題不能送出
- 新增筆記後列表立即出現
- 編輯筆記後內容立即更新
- 刪除筆記後列表同步消失
- 搜尋標題或內容都找得到
- 分類篩選可以正常切換
- 沒有符合資料時會出現空狀態
- 開啟頁面後可直接使用系統
- 重整頁面後資料仍存在
